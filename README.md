# TaxiSimulation
Tools to simulate taxi routing

## Visualization controls:
View:
- Z - zoom in
- X - zoom out
- A - Thicker drawings
- S - Thinner drawings
- UP/DOWN/LEFT/RIGHT - move camera
- Q or ESC - quit

Time:
- SPACE - pause/play
- W - accelerate time
- E - decelerate time
- R - reverse time
